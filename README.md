# 📌 Sobre o Projeto

**PyCam** é um sistema de visão computacional em tempo real desenvolvido em Python, utilizando OpenCV, MediaPipe e NumPy para detecção facial, rastreamento de movimento e processamento inteligente de câmera.

O projeto foi criado com foco em desempenho, simplicidade e expansão para futuras automações e aplicações com IA visual.

# 🚀 Como Executar

## 1️⃣ Clone o repositório

```bash
git clone https://github.com/ItzYago/pycam.git
cd pycam
```

## 2️⃣ Instale as dependências

```bash
pip install opencv-python
pip install mediapipe
pip install numpy
```

## 3️⃣ Execute o projeto

```bash
python main.py
```
